## 0. 出发点：这次失败"失败"在哪里

E1 正式主网格（Moshi R1）的 G2 三条件里，层位稳定（top-3 三种子完全一致 L29/L30/L31）与方向一致
（最小 |cos| = 0.9757）都以极大裕量通过，唯独有效秩 128 ≫ 16 失败。#30 事后诊断确认失败稳健：
最优情形（L30 逐 k 嵌套选 C）也要 57 维才过 95% 保留线，与门限相差 41 维以上。

预注册把这条失败读作"没有紧凑的低秩 floor 子空间"，出路是换位点重试或退 N1′。
**本文的核心主张是：这个解读把度量当成了对象。** 正式结果内部已经蕴含一个更精确、
也更有趣的图景，值得先用零成本的离线分析把它钉死，再决定 GPU 资源投向。

## 1. 对 G2 失败的独立再解读

### 1.1 一个已经在正式结果内部成立的数学事实：T4 的线性读出维数 = 1

T4 是二分类规格（对方 IPU 末步的 {complete, incomplete} 投射）。正式线性探针的评估分数为

```
s(x) = w · ((x − μ) / σ) + b = (w/σ) · x + const
```

AUC 只依赖分数的排序，对单调变换不变。因此**正式 T4 AUC 0.8348 本身就是沿单一方向
v\* = (w/σ)/‖w/σ‖ 的一维投影所达到的 AUC**。这不是新实验，是恒等式：全维探针的全部判别力
都压缩在 4096 维激活空间中的一个方向上。配套证据一并读：

- 两层 MLP 相对线性探针一致更差（−0.0175 至 −0.0211）→ 线性族之外没有可读的额外 floor 信息；
- 打乱标签 AUC ≈ 0.48–0.52 → 无泄漏捷径；
- 三种子（三个 90% 会话子抽样）学到的方向 |cos| ≥ 0.9757 → **这个方向本身是稳定对象**。

### 1.2 正式数字重新拼图：方向存在，但躲开了方差主轴

把 1.1 与正式 rank-k 曲线放在一起：

| 视角 | 维数 | AUC |
| --- | ---: | ---: |
| 探针方向 v\*（一维投影，恒等式） | 1 | **0.8348** |
| PCA 方差第 1 主轴 | 1 | 0.5751 |
| PCA 前 2 主轴 | 2 | 0.5704（比 k=1 还低） |
| PCA 前 8 主轴 | 8 | 0.7441 |
| PCA 前 16 主轴（38.3–44.9% 方差） | 16 | 0.7938 |
| PCA 前 79–84 主轴（54.1–60.8% 方差） | 79–84 | ≈0.8180（95% 线） |

同样是"一维"，v\* 给 0.8348，方差第一主轴只给 0.5751；前两个主轴合计几乎不含 floor 信息
（k=2 甚至低于 k=1），信息从 PC 5–8 一带才开始进入，且拖着长尾直到 ~84。三种子曲线逐点几乎重合
（最大差 ~0.001），说明这种错向本身高度稳定，不是抽样噪声。

**结论：预注册的"有效秩"（PCA-k 投影重训曲线）度量的不是 floor 信息的内在维数——那由恒等式
钉死为 1——而是 v\* 与激活方差主轴的对齐程度。G2 失败的准确表述是：**

> T4 话语权判断由残差流末端一个跨种子稳定的**一维方向**线性承载，
> 但该方向的能量散布在方差排序第 ~5 到 ~130 位的中低方差主成分带上，
> 与占据方差前列的主轴（假说：声学/内容信号）**系统性错向**。

"128 维分布式"与"1 维方向"同时为真，分歧只在度量。原报告"信号分布式地存在于数十个中等排序
主成分中"的描述没有错，但它描述的是 v\* 在 PCA 基下的坐标展开，而非读出所需的自由度。

### 1.3 为什么这比原假设更有趣

- **与叠加时代的可解释性图景吻合**：refusal direction、truth/sentiment direction 一类工作反复发现
  功能特征是单方向、且常与方差主轴不对齐；主方差被表面/内容信号占据。我们在全双工语音模型的
  对话控制信号上看到同一几何——floor 是"低幅度、方向稳定的控制平面信号"，音频内容是"高幅度的
  数据平面信号"。这是一个可标题化的发现，而非一次失败。
- **方法学贡献**：PCA 紧凑性（有效秩 ≤ k）作为"存在功能子空间"的预注册判据存在系统性失配——
  它会漏掉一切错向单方向表征。可提出"对齐秩 + 方向剔除"双度量作为替代（§2），这对使用同类判据
  的 probing 文献有普遍意义。
- **E2 未被阻塞**：H2 判据原文就是"patching/**方向注入**翻转率 ≥50%…"——方向注入本来就是一维
  操作，从未要求低秩 PCA 子空间。G2 的工程目的（保证 E2 有低维旋钮）实质上已被"方向稳定性
  |cos|≥0.976"满足。换位点重试的动机需要重新评估（§4.1）。

### 1.4 诚实清单：边界与替代解释

1. **一维充分 ≠ 一维必要**。恒等式只说明"存在一个方向达到 0.8348"，不排除激活里还有承载同样
   信息的冗余方向（PCA-16 的 0.7938 里就可能有 v\* 之外的补偿成分）。→ 用方向剔除实验（§2 B）分辨
   "单方向码"与"冗余方向簇"。两种结局都有明确叙事。
2. **跨种子 |cos| 的乐观偏差**。三种子共享 inner_val 80 会话、子抽样池重叠约 90%，方向一致性
   部分来自训练集重叠。→ 第二梯队做不相交半分训练（§3 G）给出干净下界。
3. **PCA-k 重训 ≥ 纯投影**：rank-k 曲线是重训探针，能利用 top-k 内的一切信息，因此
   E(k)（v\* 在 top-k 的能量）只会 ≤ 由 AUC 曲线粗推的份额。能量谱（§2 A）直接测 E(k)，
   把"v\* 的错向程度"与"top-k 内补偿信息"分开。
4. **本轮只覆盖 Moshi R1**。一切结论限定单模型单制式；跨模型收敛检验见 §5。

### 2.1 实验定义

**A. 方向–主轴能量谱（stage `spectrum`）** 对 L29/L30/L31 × 3 种子：训练矩阵 float64 PCA
（与 #30 诊断同解法）→ 把原始空间判别方向 v\* = w/σ（来自正式 fit 断点）分解到 PCA 基，
得能量 e_i 与累计 E(k)；报告 E(16)、E(64)、E(128)、对齐秩 alignment_rank(ρ) = min{k: E(k) ≥ ρ}，
ρ ∈ {0.5, 0.8, 0.95}；同时报告 v\* 在 PCA 张成空间内的总份额（训练行数有限时谱不满）。
自检恒等式：AUC(x_eval · v\*) 必须复现正式全维 AUC（容差 1e-3，防空间转换出错）。

**B. 读出方向的冗余结构：INLP 冗余谱 + LEACE 自检（stage `spectrum`，#35 全面修订）**
先登记一个概念性数学事实（v0.1 的"迭代均值差擦除计轮数"因此被撤销，见变更历史）：
**任何均值信号的线性擦除秩恒为 1**——擦除训练集 diff-in-means 方向后两类质心精确相等，
凸损失 + L2 正则下零权重成为全局最优，重训 AUC 必然 ≈0.5；对 1 维与 k 维（多坐标冗余）
均值信号一视同仁（LEACE 线性 guardedness 定理；合成复核 1/4/16 坐标信号全部一轮归零）。
所以"崩塌轮数"不度量信息维数，任何以它为判据的设计都恒判 α。正确的操作化分两件事：
- **INLP 冗余谱（判读用）**：每轮重训全维探针（正式 C）记录评估 AUC，剔除其原始空间
  方向，至多 4 轮；r_0 为完整重训 AUC，r_1 为剔除最优读出方向后**次优正交读出**的 AUC。
  判读量 = r_1 保留率 (r_1−0.5)/(r_0−0.5)：同质一维码（各向同性噪声）→ 低（角误差残留）；
  白化方向与均值差方向错开的多载体几何（"厚方向束"，如双尺度信号轴或上下文异质方向
  混合）→ 高。两端已由合成测试固化（<0.3 vs >0.7）。序列平台化按 indeterminate 处理。
- **LEACE 均值差擦除自检（非判据）**：擦除 d̂ 后重训，训练 AUC 必须 ≈0.5（偏离 >0.03 =
  实现有误的硬信号）；评估侧残余为"样本外均值泄漏 + 非均值线性结构"的描述量。
单次剔除 v\*/d 的数字仍登记，仅作能量谱参考（其对方向估计误差不稳健，不作判据）。
同时报告 diff-in-means 方向 d = μ₊ − μ₋ 自身：cos(d, v\*)、AUC(x·d)（零训练投影）——
d 是无监督稳健方向，refusal-direction 文献的标准做法，是 E2 转向的候选之一。

**C. 补空间探针（stage `spectrum`）** 剔除 top-k 方差主轴（k ∈ {8,16,32,64,128}）后重训：
预测 AUC 几乎不掉（k ≤ 16 时掉幅 < 0.01）。这是"方差前列对 floor 读出几乎无贡献"的直接证据，
与 A 的能量谱互为印证（A 测 v\* 在哪，C 测去掉主轴后信息还在不在）。

**D. 主轴内容鉴定（stage `spectrum`）** 用 Mimi 双通道潜表征（1024 维，与正式基线同口径同对齐）
岭回归逐个预测 top-32 PC 坐标、v\* 坐标、d 坐标（train 拟合、eval R²）：
预测"主方差 PC 被 Mimi 高度线性预测（声学内容），v\* 坐标的 Mimi R² 显著更低（超声学）"。
成立则把 H2（语义 vs 声学）与几何发现缝合成统一叙事：**模型把声学内容放在高方差通道，
把对话控制状态放在与之近正交的低方差通道**。不成立（v\* 也被 Mimi 高预测）则要求谨慎解释
+0.0659 优势的信息来源，同样有价值。

**E. 方向族结构（stage `directions`，零激活载入，秒级）** 全部 8 个二分类规格
（T1×6δ、T2、T4）× 32 层 × 3 种子的原始空间方向库（w/σ 单位化）：
- T4 逐层：跨种子 min |cos|、相邻层 |cos|、与 L29 种子均值方向的 |cos| → 方向在骨干中的
  "诞生层"与凝聚轨迹（结合 #30 的 84→68/57→66 层间秩梯度：预测末几层方向渐趋固定、对齐渐紧）；
- cos(T1_dδ, T4) 随 δ 的曲线（各层）→ 前瞻话轮预测与当前 floor 判断是否同一方向：
  高且平缓 = "前瞻是状态判断的时间外推"；随 δ 旋转离开 = 独立前瞻通道（planning 式表征，更有趣）；
- T2 与 T4/T1 的方向关系（T2 无信息优势，但方向仍可能落在同一族内）。
- 附：T1 优势随 δ 扩大（+0.0068→+0.0238）而 Mimi 衰减更快——按 δ80→δ800 端点粗算，
  探针超机会 AUC 的经验半衰期 ≈ 2.5 s，Mimi ≈ 2.0 s。内部前瞻比声学表面线索更持久，
  方向族分析将回答"这份持久性是否由同一 v\* 承载"。

**F. 事件锁定方向轨迹（stage `trajectory`，仅评估集 100 会话，~5 GB 载入）** 把评估集 L29
完整时间轴投影到 probe_meanseed（转向包种子均值方向）与 diffmeans，以对方 IPU 末步为 0 点
取 ±25 步（±2 s）窗，按 complete/incomplete 分组做会话级 cluster-bootstrap 均值带：
**直接可视化"floor 判断信号"的时间展开**。显著性判定（#35 修订）：51 个偏移逐点 95% CI 的
"首次排除零"在零效应下假阳性率接近 1，故起点判定采用 **studentized sup-t 同时置信带 +
连续 ≥3 步显著**（与 `e1x/trajectory.divergence_step` 的持续性约束对齐；零效应假阳性已由
合成测试固化为受控）。若分离早于 IPU 末点数百毫秒，说明模型在"预判着听"，与 T1 前瞻
结果闭环；这也是论文的核心插图候选。对照方向：方差第 1 主轴与固定种子随机方向。

**G. 转向向量包（stage `spectrum` 层内三种子齐备后落盘）** 每层输出
`steering_L{29,30,31}.npz`，**schema 与 E1-X 的 `e1x-directions-v1` 完全兼容**（键：
`probe_s{seed}`/`probe_meanseed`/`diffmeans_s{seed}`/`diffmeans`/`random_r{i}`；
`__meta__.proj_std` 提供 α·s_v 剂量标定，per-seed 方向取训练集口径、种子均值与随机方向
取评估集口径并在 meta 注明）。符号约定：**+v 指向 T4 label=1**，由类条件投影统计显式核验、
不做静默翻转（#35 中风险 5）。`wp_e2_lite_plan.py --directions-npz` 可直接指向本包——
E1-X geometry 与几何解剖两条支线互为 E2-lite 的备援方向源。E2 先导协议见 §4.2。

### 2.2 判读矩阵（预测登记，防事后解释自由度）

| 观测量 | 结果 α | 结果 β |
| --- | --- | --- |
| E(16)：v\* 在前 16 主轴的能量 | 低（<0.5）→ 错向主张成立 | 高（>0.8）→ 与 rank-k 曲线矛盾，须查实现 |

| T4 逐层方向 | L28→31 相邻 \|cos\| 均 ≥0.9 且对齐秩@0.95 L31≤L29 → 凝聚动力学 | 任一相邻 <0.7 → 读出通道非固定 |

任何一列取 α 或 β 都有明确后续；没有"失败"分支。均值差擦除自检不在判读矩阵内
（其结果由数学决定，只校验实现）。八行全部由 `finalize` 的 `verdict_matrix` 自动判定。

## 4. 对预注册路线与论文叙事的影响

### 4.1 G2 与换位点重试的重新定位

- G2=fail 照旧登记，不改门限、不重开裁决——本支线是解释层，不是翻案层。
- 预注册的"L29–L31 attention/MLP 输出换位点重试"需要重跑前向缓存（数百 GB 级 GPU 任务）。
  其动机是"残差流不紧凑，也许子模块输出更紧凑"。**若 A/B 证实"1 维错向"图景（E(16) 低 +
  INLP r₁ 保留率 ≤0.3），换位点对'找紧凑子空间'的边际价值即告消失**——旋钮已在手（v\*），
  紧凑性失败只是度量失配。建议：换位点降级为条件执行（K′），仅当 B 出现"厚方向束"分支
  （INLP r₁ 保留率 ≥0.7）时再投入，用于判断多载体读出究竟分布在 attn 还是 MLP 通路。
  GPU 卡时优先让给 PersonaPlex 缓存（§5）。

### 4.2 E2 的方向注入升级（先导协议草案，正式操作化另行登记）

- 转向向量：v\*_mean（主案）与 d_mean（稳健对照），来自 §2 G 的转向包；
  α 以训练集类条件投影 σ 为单位，扫 {±0.5, ±1, ±2, ±4}σ。
- 注入位点：L29 残差流（主）、L30/L31（复核）；时机锁定对方 IPU 末点邻域（F 的轨迹分离窗
  给出注入窗口的实证依据）。
- 判据沿用 H2 原文（翻转率 ≥50%、高出范数匹配随机方向 ≥30 pt、剂量单调 ρ>0.8、
  内容方向对照无效）——**原判据文本无一字需要修改**，只是"方向"的来源从"低秩子空间主轴"
  换成"探针判别方向"，且后者有跨种子 |cos|≥0.976 的稳定性背书。
- 行为终点在 R2/R3 制式（本机 GPU），与转向包解耦；先导可只跑 20 会话验证翻转方向性。

### 4.3 论文叙事分支（取代原"不满足就退化"的单行线）

- **N0′（主案，若 B 取"一维码"）**："Floor control rides a stable, variance-minor direction"——
  全双工语音模型把话轮让渡判断压缩在残差流末端一个与方差主轴错向的一维方向上；
  声学内容占据主方差（D 成立时）。贡献 = 发现 + 因果（E2）+ 方法学警示（PCA 有效秩失配）。
- **N0″（若 B 取"厚方向束"）**："A redundant low-dimensional bundle"——floor 信息冗余分布于
  少量非主轴方向，单方向可读但非必要；E2 改测方向束的联合消融。
- **N1′（保底，两梯队全部无聊时）**：按预注册转入，保留 T4 强可读性、层位稳定、方向一致
  作为核心描述性结果。
- 无论分支：T1 的"优势随 δ 扩大 + 内部半衰期更长"独立成节（预判性听觉），
  T3/T5 与 Mimi 打平作为"增益谱 = 抽象度谱"的对照证据（可由声学直接解出的目标上模型无增益，
  需整合对话结构的目标上才有增益——这本身是 H2 的第一块拼图）。

## 5. 资源与五模型矩阵的优先级建议

1. **CPU/闲时单卡**：本文第一梯队（几何解剖），出全部判读矩阵结论——它决定 E2 与换位点的走向。
2. **GPU 主力**：PersonaPlex-7B E1 缓存与探针网格（同家族 RQ-Transformer，L1 架构）。
   跨模型预测（探索性登记）：若"末端层一维错向方向"在 PersonaPlex 复现，H5 收敛叙事升级——
   "错向的一维 floor 方向"本身可能就是跨模型收敛解，这比原计划的 CKA 相似度更锋利。
3. **随后**：MiniCPM-o / Freeze-Omni（L2 架构）与 dGSLM；R2 制式与跨语料泛化按原计划推进。
4. E2 先导（§4.2）在 PersonaPlex 缓存排队间隙即可插跑（20 会话、单卡、小时级）。
