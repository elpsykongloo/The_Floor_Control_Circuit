# MiniCPM-o 4.5 上下文最终结论

日期：2026-07-21

## 最终结论

1. **工程默认长度取 600 秒。** 在三条不同录音、每条两个 lane 的完整因果交互中，
   600 秒的均衡听说为 `12/12`，远端记忆为 `6/6`，同位置近期记忆为 `6/6`，无事实
   负对照为 `0/6`。三条录音分别保持满分。
2. **任务退化过渡带为 752–816 秒。** 752 秒开始出现内容依赖错误；768 秒时听说降至
   `9/12`、远端记忆降至 `4/6`；800 秒起听说连续异常；816 与 832 秒的记忆高位置
   控制连续失效。
3. **两条任务轨道的共同可靠检查点为 784 秒。** 该数字来自持续失效判据。752 秒后
   已经观察到零星能力下降，所以 784 秒不能作为“无退化”保证。
4. **2000 秒完整上下文能力没有实证保证。** 密集实验表明同位置近期内容在 2600 秒
   等效位置仍可稳定运行，但完整长历史从 768 秒起已经持续改变模型决策。局部位置
   外推不能证明远端语义、话轮控制和记忆仍然完整。
5. **正式规格保持 40960 个位置。** 在人工饱和负载的 32 位置/秒条件下，可容纳 1279
   个完整一秒单元。自然全双工交互的位置消耗随模型发言而变化，秒数只能结合实际
   KV 位置解释。

因此，项目统一采用以下口径：

- 需要尽量避免可见退化：**600 秒**；
- 研究环境可探索的偏积极上限：**736 秒**；
- 描述性共同可靠检查点：**784 秒**；
- 进入持续失效的区间：**800–816 秒**；
- **不宣称 2000 秒无退化。**

## 合并后的证据

| 证据 | 最终发现 | 解释 |
| :--- | :--- | :--- |
| 正式配置 | `max_position_embeddings=40960` | 模型声明的位置上限 |
| 密集绝对位置对照 | 同位置近期后缀实测到 2600 秒等效位置仍稳定 | 只支持局部位置外推 |
| 密集完整历史 | 768 秒起出现持续决策分叉 | 远端历史已经显著改变行为 |
| 因果均衡听说 | 600、736 秒满分；752 秒起下降；800 秒起持续异常 | 支持 600 秒工程上限 |
| 因果远端记忆 | 600、736 秒满分；816、832 秒高位置控制持续失效 | 总体任务容量在此区间下降 |

所有正式探针和模型回复都会进入后续 KV 历史。本实验测量完整听说交互历史，用户音频
只是历史的一部分。记忆查询的首 token 固定为 `<|speak|>`，其余 token 保持官方
贪心生成，因此记忆结果表示条件检索能力；自然话轮决策由独立的均衡听说轨道测量。

## 适用限制

- 证据来自三条英语录音、两个显式听说最小对和有限英语口令；
- 每个记忆检查点只有 6 个观测，录音与探针之间存在聚类；
- 结论适用于 MiniCPM-o 4.5、当前系统提示和贪心参数；
- 816 秒后的近期高位置控制也会失效，当前结果不能唯一归因为纯远端遗忘。
